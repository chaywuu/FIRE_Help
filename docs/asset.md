# 📈 资产和负债

资产写在最前面，因为我觉得是FIRE的基础，你有多少钱才能决定你能否FIRE，决定你以哪种方式FIRE。

当然，**资产也并不是现金**。

你需要好好盘点自己的资产，包括现金、存款、投资等。在这个过程中，你需要知道哪些是资产，哪些是负债。就如信用卡，它不是你的资产，是别人对你的授信，你使用的每一分钱，会在下一个账单日具象化，你其实在透支你的未来。

**我并不建议每个人都使用信用卡**。

## 新增资产

在资产页可以新增资产或者负债，资产负债会根据母类（例如现金类资产）归集汇总：

![资产图片](/images/asset/asset1.jpg)

资产类：
* 现金类资产：可以是储蓄卡、现金、微信钱包等类现金或者现金资产，它们的价值是可以直接转换为现金的。
* 充值类资产：可以是通过充值获得的资产，例如公交卡、会员卡等。
* 投资理财：可以是股票、债券、基金等金融资产。
* 应收：即借出的资金。
* 固定资产：房产、车产等固定资产。

负债类：
* 信用类资产：外部资产提供的授信额度，能在授信额度下提前透支，例如信用卡、花呗等。
* 应付：即借入的资金。
* 贷款：京东金条、微粒贷等信用类贷款，或房贷、车贷等固定资产类贷款。

每个资产填写内容或有不同，根据要求填写即可。
