# 📅 预算管理

通过预算日历的日期切换年份，预算日历中展示对应年份下的每月总和已用情况：

![预算图片](/images/budget/budget1.jpg)

点击不同的月份，总预算卡片会展示对应月份的**总预算**：

![预算图片](/images/budget/budget2.jpg)

## 编辑总预算

在月预算和年预算模式下，点击编辑图标可对总预算进行编辑：

![预算图片](/images/budget/budget3.jpg)

## 编辑分类预算

在月预算和年预算模式下，点击分类名称可对分类预算进行编辑，分类预算也支持二级分类的预算编辑：

![预算图片](/images/budget/budget4.jpg)

## 查看已用

点击查看图标可查看对应分类的已用情况：

![预算图片](/images/budget/budget5.jpg)

## 预算推荐

根据历史数据，总预算和分类预算卡片会展示推荐预算数据，可以根据实际情况进行调整：

![预算图片](/images/budget/budget6.jpg)
