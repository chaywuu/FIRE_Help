# 🤔 常见问题

## FIRE火记 支持什么平台？

目前仅支持PC端，后续会逐步迭代手机H5端和app端。

## FIRE火记 会偷看我的数据么？

不会。

FIRE火记提供最为不敏感的信息（邮箱）进行注册，你记录的每一条数据都属于你自己。

而且你要知道，记录的所有信息，可以都是真的，也可以都是假的，数据对于其他人是没有意义的，数据仅对你有意义。

因此，希望你能在FIRE火记中真实的记录每一条账单，因为它不仅是钱的流动，更是生活细节的记录。

而且，FIRE火记会通过技术和规则约束我的行为：

* 技术方面，FIRE火记采用业内标准的加密传输协议，并严格制定了数据安全标准，具体可以参考[安全套接字层 (SSL) 技术](https://baike.baidu.com/item/%E5%AE%89%E5%85%A8%E5%A5%97%E6%8E%A5%E5%B1%82/9442234) 。
* 规则方面，就如上面所说，我不知道你是谁，也不知道你的数据是真是假，因此，我不会偷看你的任何数据。

👉 [点击查看隐私政策](https://help.fireapp.me/privacy.html)

## FIRE火记 可以买断吗？

不可以。

因为 FIRE火记 是云服务，随着使用时长增加，存储空间也会不断增长。如果买断将来会导致入不敷出，具体可以参考 iCloud 或者各种云盘服务。

不能买断其实是为了让我们的服务能更好更久的服务下去。

👉 [点击查看会员简介](https://help.fireapp.me/pro.html)
